UCI Grabber Maia3 direct-download notice
========================================

UCI Grabber's GitHub release does not contain Maia3, CPython, PyTorch, NumPy,
python-chess, or Maia checkpoint bytes.  When a user chooses a Maia model,
UCI Grabber retrieves each hash-pinned artifact directly from its upstream
publisher and assembles a private, portable UCI package on that user's
computer.  UCI Grabber does not relicense any retrieved material.

The only Maia-related native asset published by UCI Grabber is its small
Apache-2.0 process launcher.  The launcher starts the locally assembled Python
environment through a process boundary.  Its complete source and the Python
entry-point source are in the tagged UCI Grabber repository.

Runtime dependencies
  The portable CPython build is retrieved from the immutable Astral
  python-build-standalone release.  Python packages are retrieved as exact
  wheels or source archives from PyPI or PyTorch's official CPU wheel index.
  Each component retains its own license and notices.  The signed release
  catalog records every URL, byte count, SHA-256 digest, and destination.

Checkpoints
  Checkpoint bytes are retrieved only after the user selects a model.  Every
  download is from an immutable Hugging Face revision and is checked by byte
  count and SHA-256 before it can be used.
